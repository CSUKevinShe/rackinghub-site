# 竞品分析技能

2. **Mecalux** (mecalux.com) - 欧洲最大仓储货架制造商
3. **Interlake Mecalux** (interlakemecalux.com) - 北美市场

## 分析维度

### 1. SEO 策略
- Title/Meta 写法模式
- 核心关键词布局
- 博客内容矩阵
- 内链网络结构
- 外链建设（通过 Ahrefs/Semrush）

### 2. 内容策略
- 产品页信息架构
- 内容深度（字数、图片、视频）
- 博客更新频率
- 案例研究/白皮书
- 多语言支持

### 3. 技术实现
- 网站速度（PageSpeed Insights）
- 移动端体验
- 结构化数据使用
- CDN 和性能优化

### 4. 转化优化
- CTA 位置和文案
- 表单设计（字段数量）
- 信任背书（认证、客户 logo）
- 社交媒体集成
- 在线聊天工具

## 执行步骤

1. **访问竞品网站**（使用 browser_navigate）
2. **抓取关键页面**（首页、产品页、博客、About）
3. **提取 SEO 要素**（Title, Meta, H1, 关键词）
4. **分析内容结构**（字数、模块、CTA）
5. **对比分析**（与 RackingHub 的差异）
6. **生成报告**（优势、劣势、可借鉴点）
7. **制定行动计划**（优先级排序）
